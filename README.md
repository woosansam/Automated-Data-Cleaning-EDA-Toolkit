# Data Analyzer Toolkit

A lightweight, open-source Python toolkit designed to automate the repetitive tasks of data cleaning and Exploratory Data Analysis (EDA).

## Purpose
In data science projects, data cleaning and initial exploration are time-consuming. This toolkit provides a modular solution to streamline these processes, helping researchers and students focus on insight generation rather than boilerplate code.

## Key Features
- **Automated Data Cleaning:** Handles missing values and optimizes data types automatically.
- **EDA Automation:** Generates summary statistics and essential visualizations (histograms, correlation heatmaps) for numerical data.
- **Modular Structure:** Easy to integrate into existing data pipelines.
